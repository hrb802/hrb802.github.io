写在前面

这是我第一次用Hexo搭建博客，搭建过程中参考的是 https://fe32.top/ 这个博主的内容很详细也很丰富，但是自己写的过程中还是遇到了很多问题，至于怎么解决。。毕竟这是一个AI时代。
起初想搭建这个博客的理由很简单，就是因为它好看，但既然被我搭建起来了（虽然还没有全部搭建好）那我会在上面写点东西。我并不是一个擅长编程的人，对于计算机的知识只能说是一星半点，不过我还是想通过这个博客来记录我自己的学习过程，让自己不要太清闲。
愿我们所愿皆所得

09月05日

HTML与CSS部分快要学完了，一直没有时间更新HTML基础标签与CSS样式（一）系列，我看才更新到盒子模型，但是一点都没有时间啊。
这段时间我就是把老师单拎出来的一些值得关注的部分在这里罗列了上来，比如常见老版本浏览器兼容处理，过一段时间更新一下编程规范 HTML + CSS，感觉这个还比较重要，而且有助于工作当中的书写规范和维护方便。

margin和padding 设置为10%(百分比) 是按照父容器的宽度来计算的,
比如

此时,wrap的左边内边距就是来自父级container的宽度100px的10%,也就是10px.
这个特性可以实现高度等比自适应布局,解决图片渲染导致高度不统一发生的频闪问题
如果是padding-top的话,如果设置为10%,也是按照父级的宽度进行计算的.

100vh 整体视图窗口的高度
100vw 整体视图窗口的宽度
使用vw\wh就不需要设置

body {
height: 100%;
}
