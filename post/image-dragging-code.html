jQuery节点查询

本段使用HTML源码片段

引用的库为 jQuery库的1.12.1版本

$wrap.children('p');  获取队形$对象所有节点的所有子元素 传参 限制 类型String = 选择器

$lis.eq(0);    eq方法 参数index 下标 {number} 返回一个新的指定下标的$DOM对象

$lis.get(0);   get方法 参数index 下标{number} 返回对应下标的原生DOM节点对象

$lis.first();  .eq(0), 第一个$DOM对象

$lis.last();   .eq($lis.length - 1) 最后一个$DOM对象

$lis.filter('.active');    过滤并返回对应规则的$DOM

类似some() , 返回bollean
$lis.hasClass('.active');  判断$DOM对象是否包含某个类名, 只要DOM集合中有一个对象包含对应类名, 返回true, 一个都没有包含对应类名, 返回false

$lis.eq(0).next('.active')   如果next()参数条件符合,返回对应$DOM对象的下一个$节点对象 参数 选择器条件, 如果下一个节点与参数条件不符合, 返回空的$DOM对象

$lis.eq(1).nextAll();  返回对应$DOM队形后续所有的兄弟节点对象

$lis.eq(4).prev();  对应$DOM对象前面一个兄弟节点对象

$lis.eq(4).prevAll();  对象$DOM对象前面所有的兄弟节点对象

$lis.eq(2).siblings();  获取除了自身之外的同层父元素的$DOM对象 参数 选择器 用于筛选

$text.parent('.active');  返回每一个$DOM对象的父元素的$DOM对象

$text.not('#not'); 返回$DOM对象集合中排除复合not条件参数的其他$DOM对象

$lis.length; 返回$DOM对象集合的数量

jQuery的多数DOM操作获取的方法都会返回一个DOM对象
.get() 返回原生DOM方法
.hasClass() 返回bollean

jQuery属性操作

  attribute 标签属性  
  attr() 获取标签属性
    一个参数(非JSON)String  获取
    两个参数      设置
    一个JSON参数  多项设置
    操作标签自定义属性 和 非功能文本属性

  prop
    一个参数(非JSON)String  获取
    两个参数      设置
    一个JSON参数  多项设置
    操作标签默认属性 和 默认功能属性

  removeAttr 删除自定义属性, 可以删除标签上任何属性(属性名+属性值)

  removeProp 更保守 标签自身属性不会删除 只会设置为undefined

  (add   remove    toggle) Class 
  toggleClass 有则删 无则加
    
  
  val()   =>   .value
  text()  =>   innerText
  html()  =>   innerHTML
  传参   --  设置
  不传参 --  获取

jQuery 样式操作

css() 设置样式  
  一个参数 获取
  两个参数 设置
  一个json参数 批量设置

offset()  参考原点是document文档左上角
  不传参数类似于getPosition();
  传参 {top, left} 类似于setPosition()

position()  参考原点是定位父级左上角  
  类似 .offsetTop .offsetLeft
注意: position() <strong>没有参数 </strong>, 只能获取 不能设置

scrollTop 垂直滚动条滚动高度 获取 设置。
  $(document).scrollTop()
  不推荐 $('body').scrollTop()

  推荐 $(document).scrollTop()

scrollLeft  水平滚动条滚动高度

读/写  width() height() 单纯设置和获取 元素content内容的宽度高度 

读   innerWidth/height() 获取元素content+padding宽度/高度 

读   outerWidth/height() 获取元素content+padding+border 宽度/高度 

读    outerWidth(true)/height(true) 获取元素content+padding+border+margin 宽度/高度

在设置位置的时候, 推荐使用css直接设置样式, 不推荐使用offset

jquery 事件

dom.on事件名称 = function()

dom.addEventListener('事件名称', function(){}, false)

ready() => window.onload

focus() => onfocus

click() => onclick

hover() => mouseover() + mouseout()

trigger 执行DOM的默认事件行为

推荐用原生DOM的事件方法
$('form')[0].submit()
$('form').get(0).submit()

on('事件名称', '委托选择器', 事件句柄) off ('事件名称')
