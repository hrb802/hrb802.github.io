什么是Markdown

Markdown 是一种轻量级标记语言，使用纯文本格式编写文档，然后转换为有效的HTML文档。

优点：

  • 易读易写
  • 纯文本，兼容所有文本编辑器
  • 可转换为HTML、PDF等多种格式
  • 被GitHub、GitLab等平台广泛支持
