Python基础学习

发表于2025-06-21|更新于2025-07-09|学习记录
|总字数:2.4k|阅读时长:9分钟|浏览量:
在这里默认已经搭建好python环境以及编译器，就不过多描述了。

基础语法

变量

定义变量

  1. 定义变量
    在 Python 中，变量不需要提前声明类型，直接给变量赋值即可完成定义。变量名遵循一定的命名规则，只能包含字母、数字和下划线，且不能以数字开头，同时不能使用 Python 的关键字。
    查看Python的关键字：
  2. 变量赋值
    变量赋值可以使用等号（=）进行，将等号右侧的值赋给左侧的变量。

数据类型

数字

  1. 整数
    整数是没有小数点的数字，包括正数、负数和零。在Python中可以表示任意大小的整数，没有长度限制。
  2. 浮点数
    浮点数是带有小数点的数字，用于表示小数。在Python中浮点数的精度是有限的，不能表示所有的小数，可能会有舍入误差。

字符串

  1. 字符串定义
    字符串是由字符组成的序列，可以使用单引号（’）、双引号（”）或三引号（’’’或”””）定义。
  2. 字符串拼接
    字符串可以使用加号（+）进行拼接，也可以使用乘号（*）重复拼接。

流程控制

条件语句

  1. if语句
    if语句用于判断条件是否成立，如果成立则执行相应的代码块。
  2. if-else语句
    if-else语句用于判断条件是否成立，如果成立则执行if代码块，否则执行else代码块。
  3. if-elif-else语句
    if-elif-else语句用于判断多个条件是否成立，如果成立则执行相应的代码块。

循环语句

  1. for循环
    for循环用于遍历序列（如列表list、元组tuple、字符串str），依次取出序列中的每个元素，并执行相应的代码块。

    for循环遍历字符串

    for循环遍历字典

    for循环遍历字典的键值对

    for循环遍历列表的索引

  2. while循环
    while循环用于重复执行一段代码，直到条件不成立。

  3. break语句
    break语句用于跳出循环，当循环执行到break语句时，会立即跳出循环。

  4. continue语句
    continue语句用于跳过当前循环，当循环执行到continue语句时，会跳过当前循环，继续执行下一次循环。

  5. pass语句
    pass语句用于占位，当语法上需要语句，但程序不需要任何动作时，可以使用pass语句。

常用模块

Python标准库

os模块

  1. os模块介绍
    os 模块是 Python 标准库中的一个重要模块，它提供了一系列与操作系统进行交互的函数，可用于处理文件和目录、管理环境变量等操作。这使得 Python 程序能够跨不同操作系统（如 Windows、Linux、macOS）运行，增强了代码的可移植性。

  2. os模块常用函数

    • os.getcwd()：获取当前工作目录
    • os.chdir(path)：改变当前工作目录
    • os.listdir(path)：返回指定目录下的所有文件和目录名
    • os.mkdir(path)：创建单层目录
    • os.makedirs(path)：创建多层目录
    • os.remove(path)：删除文件
    • os.rmdir(path)：删除单层目录
    • os.removedirs(path)：删除多层目录
    • os.system(command)：运行系统命令
    • os.environ：获取环境变量
    • os.path模块：处理文件路径
