Hexo创建博客的front-matter内容详细信息（基本元数据配置）

发表于2025-07-25|更新于2025-07-25|学习记录
|总字数:814|阅读时长:2分钟|浏览量:

基础信息

id: ******
文章的唯一标识符，可用于区分不同文章，在一些场景下方便管理和引用特定文章。

title:****
文章的标题，会显示在博客文章列表、文章页面顶部等位置。

分类与标签

tags:

文章的标签，以数组形式存在。标签用于对文章进行多维度分类和快速检索，方便用户查找相关主题的文章。

categories:

文章所属的分类，同样以数组形式存在。与标签不同，分类通常具有一定的层级结构，用于对文章进行系统性的归类。

页面展示相关

description: *******************
文章的描述信息，一般会显示在搜索引擎结果页、文章列表预览等位置，帮助用户快速了解文章内容。

cover: *******（图片链接）：
文章封面图的链接，会显示在文章列表页，起到吸引用户点击的作用。
（比如我的：https://cdn.jsdelivr.net/gh/hrb802/Images@main/Chuyin.png）

top_image: *******（图片链接）
文章顶部大图的链接，通常显示在文章页面的顶部位置。

目录设置

toc: true
是否开启文章目录功能。true 表示开启，开启后会在页面上显示文章目录。

toc_number: true
目录项是否显示序号。true 表示显示，会给每个目录项添加序号。

toc_style_simple: true
是否使用简洁风格的目录样式。true 表示使用。

版权信息

copyright: 文章的版权信息，包含以下子字段：

    author:*****         //文章作者的名字。
    link: 'https://*******.github.io'         //作者个人主页的链接。
    url: 'https://******.github.io/<id>/'         //文章的链接。
    info:                //版权声明信息

示例：
copyright:
author: ******
link: ‘https://.github.io’
url: ‘https://.github.io/blog1474/‘
info: >-
本博客所有文章除特别声明外，均采用 CC BY-NC-SA 4.0 协议，转载请注明出处。
AI生成项目
这里表明文章采用 CC BY - NC - SA 4.0 协议，转载需注明出处。

功能开关

mathjax: true
是否开启 MathJax 数学公式渲染功能。true 表示开启，开启后可以在文章中使用 LaTeX 语法编写数学公式。

katex: true
是否开启 KaTeX 数学公式渲染功能。KaTeX 是另一种数学公式渲染引擎，和 MathJax 类似，true 表示开启。

aside: true
是否显示侧边栏。true 表示显示。

aplayer: true
是否开启 APlayer 音乐播放器功能。true 表示开启，开启后可以在文章中嵌入音乐播放器。

highlight_shrink: true
代码高亮块是否默认收缩。true 表示默认收缩，用户点击后可展开。

其他信息

abbrlink: **********
文章的缩略链接，用于生成短链接，方便分享和记忆。

date: 年-月-日 时:分:秒
文章的创建日期，会显示在文章页面上，用于记录文章创作时间。
